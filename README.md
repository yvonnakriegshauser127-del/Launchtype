# Pipeline Tool

Интерактивный пайплайн для отслеживания этапов процесса с возможностью создания майлстоунов.

## Технологии

- React 18
- Vite
- TypeScript
- Ant Design 5
- React Router

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:3000`

## Маршруты

- `/pipeline` - Основная страница с пайплайном (standalone режим)
- `/pipeline/embed` - Режим для встраивания в iframe (embed режим)

## Функциональность

### Этапы пайплайна

1. Новая идея
2. Работа с поставщиками
3. Логистика
4. Разработка
5. Предпроверка
6. Акции

### Майлстоуны

- Создание майлстоунов в каждом этапе
- Указание названия майлстоуна
- Установка даты начала (обязательно)
- Установка даты окончания (только для завершенных майлстоунов)
- Отметка о завершении майлстоуна
- Отображение дат при фокусе на майлстоуне

## Локализация

Поддерживаются три языка:
- Русский (по умолчанию)
- Английский
- Украинский

Язык выбирается через `localStorage.setItem('language', 'ru'|'en'|'uk')`

